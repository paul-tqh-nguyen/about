Netflix Actor Network Hashing Comparison

Evaluation of Graph Data Representations for Algorithms Performed on the Netflix 2019 Actor Collaboration Network

This work uses the actor collaboration network generated from the Netflix movies of 2019 to show how hash-table representations for graph data (despite their approximately constant-time cost for accesses and mutations) can lead to slow performance. We demonstrate this on the task of calculating the Kevin Bacon index of each actors.
